# async 비동기 함수

async를 함수앞에 붙이면 그 함수는 자동으로 promise반환

비동기 함수 - 언젠가 완료될 작업을 표시 : promise
js의 경우 한줄씩 순서대로 실행하는데 네트워크 요청,파일 읽기, 타이머 같은 일은 시간이 걸린다.

따라서 시간이 걸리는 작업의 경우 비동기 방식으로 처리한다.

# 토큰을 처리하는 과정

: 백엔드(서버) API에 아이디와 비밀번호를 보내고,
서버가 인증 성공의 증표로 토큰을 발급해준다.

: 프론트 입장에서의 핵심은 :
어떻게 API를 호출하고, 응답으로 받은 토큰을 어디에 저장할지.

**1.API에 PST방식으로 전송**

- react앱이 fetch나 axios같은 라이브러리를 사용해 정보를 백엔드 서버의 로그인 api에 post 방식으로 전송

서버 인증 및 토큰 발급 : 백엔드 서버는 받은게 유효한지 확인
성공시 서버는 토큰을 생성하여 JWT -> API응답의 본문에 담아 react앱으로 다시 보내줍니다.

토큰저장 : React 앱은 성공 응답으로 받은 토큰을 브라우저 저장소에 저장.
이 토큰은 앞으로 서버에 데이터를 요청할때 나는 인증된 사용자인거를 증명가능

### 추가 개념

**API 호출** : 백엔드 서버와 대화하기 위해서 API 호출를 해야됨
fetch, axios는 이 api호출을 쉽게 하도록 도와주는 도구

- fetch :
  기본 내장 전화기, 응답을 json형태로 변환하는 등 몇가지 추가 작업을 직접 해줘야한다. 이미 내장되어 있다

- axios : 스마트 기능이 추가된 앱, fetch와 같지만 많은 편의 기능을 제공한다. 그래서 npm install axios명령어로 프로젝트를 설치해야한다.
- 자동 데이터 json형식으로 변환

**HTTP 요청 방식**

1. POST : 상자 BODY안에 숨겨서 보냄 "보낼때"
2. GET : 엽서 보내기 -> 목적 : 서버로부터 데이터를 조회하거나 가져올때 사용 : 조회/가져올때
